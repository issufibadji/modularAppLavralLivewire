## Contexto Atual

Este projeto NÃO está sendo migrado de um sistema legado existente.

A aplicação foi reiniciada do zero em uma nova base Laravel 12, com objetivo de:

- eliminar dívida técnica anterior
- corrigir problemas estruturais (PSR-4, módulos duplicados, acoplamento)
- adotar arquitetura modular limpa desde o início
- reconstruir funcionalidades com base no comportamento do sistema anterior, mas sem reaproveitar código legado

O sistema antigo serve apenas como referência funcional (regras de negócio), não como base técnica.
